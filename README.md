## Installera Ansible
 1. Installera brew.  ``` /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" ```
 3. Installera Ansible. ``` /bin/bash brew install ansible ```
